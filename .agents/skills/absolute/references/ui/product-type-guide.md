# Product Type Design Guide

Maps product types to concrete design recommendations - the bridge between
"I'm building X" and knowing exactly which style, colors, fonts, and layout to use.

## How to use this reference

1. Identify the product type closest to the user's project
2. Apply the style, colors, typography, and landing pattern from the table
3. Follow the key rule - it captures the single most important design constraint
4. Combine with other reference files (typography.md, color-and-theming.md, landing-pages.md) for implementation details
